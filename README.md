# Restaurant AI Assistant

Restaurant AI Assistant is an AI-powered chatbot built using Streamlit that helps restaurants automate customer interactions.

The system can answer menu questions, provide opening hours, assist with bookings, and export order or booking data.

## Features

- AI chatbot interface
- Restaurant configuration panel
- Menu and opening hour queries
- Booking and order export to CSV
- Streamlit web interface

## Technologies

- Python
- Streamlit
- Large Language Models (LLMs)
- Prompt Engineering

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
